# Buenas Prácticas

Maximiza el valor de Sentinel Express con estas recomendaciones.

1. **Revisiones Semanales:** No esperes al cierre de mes. Valida tus facturas recibidas semanalmente para solicitar correcciones a tiempo.
2. **Historial de Listas Negras:** Actualiza la base de datos de EFOS regularmente en la sección de Configuración para detectar proveedores riesgosos.
3. **Resguardo de XML:** Guarda siempre los archivos XML originales, ya que son el único documento con validez legal, no el PDF.
4. **Verificación Dual:** Si un CFDI marca alerta, utiliza el botón de "Consultar SAT" para asegurar que el estatus de cancelación no ha cambiado.
