# Sentinel Express Dashboard - Exploración de Diseño

## Respuesta 1: Minimalismo Corporativo Moderno (Probabilidad: 0.08)

**Movimiento de Diseño:** Corporativo Minimalista con influencias de Bauhaus

**Principios Centrales:**
- Claridad absoluta: cada elemento tiene propósito fiscal
- Jerarquía tipográfica fuerte: títulos sans-serif en peso 700, cuerpo en 400
- Espaciado generoso: respiración visual entre secciones
- Precisión geométrica: bordes limpios, alineaciones perfectas

**Filosofía de Color:**
- Paleta monocromática con acentos: gris oscuro (foreground) sobre blanco puro
- Acentos en azul profesional (RGB 30, 90, 168) para estados críticos
- Verde (RGB 34, 197, 94) para USABLE, rojo (RGB 239, 68, 68) para NO USABLE
- Justificación: transmite confianza, profesionalismo, exactitud fiscal

**Paradigma de Layout:**
- Grid de 12 columnas con márgenes amplios
- Sidebar izquierdo para navegación secundaria
- Tarjetas con sombra suave (0px 1px 3px rgba(0,0,0,0.1))
- Tablas con filas alternadas en gris muy claro

**Elementos Distintivos:**
1. Indicadores de semáforo fiscal: círculos de 24px con iconografía clara
2. Barras de progreso horizontales para cumplimiento de validaciones
3. Badges con código de estado (CRÍTICO, NO DEDUCIBLE, USABLE CON ALERTAS)

**Filosofía de Interacción:**
- Hover: cambio de fondo a gris 100, cursor pointer
- Click: transición de 200ms a color primario
- Tooltips en gris oscuro con texto blanco

**Animación:**
- Entrada de tablas: fade-in de 300ms
- Cambio de estado: transición suave de color en 150ms
- Carga de datos: skeleton screens en gris 200

**Sistema Tipográfico:**
- Display: Poppins Bold 32px para títulos principales
- Heading: Poppins SemiBold 18px para subtítulos
- Body: Inter Regular 14px para contenido
- Mono: IBM Plex Mono 12px para UUID y RFC

## Respuesta 2: Diseño Forense Técnico (Probabilidad: 0.07)

**Movimiento de Diseño:** Tech Noir con inspiración en herramientas de análisis forense

**Principios Centrales:**
- Densidad de información: máximo contenido sin saturación
- Contraste alto: fondo oscuro, texto claro para legibilidad
- Capas visuales: múltiples niveles de información revelados progresivamente
- Precisión técnica: cada píxel cuenta

**Filosofía de Color:**
- Fondo: gris oscuro (RGB 23, 23, 23)
- Texto primario: blanco puro
- Acentos: verde neón (RGB 0, 255, 136) para validaciones exitosas
- Alertas: naranja (RGB 255, 140, 0) para advertencias
- Errores: rojo brillante (RGB 255, 71, 87)
- Justificación: evoca herramientas de análisis técnico, transmite urgencia y precisión

**Paradigma de Layout:**
- Tres columnas: panel lateral izquierdo (filtros), contenido central (tabla), panel derecho (detalles)
- Gráficos de línea para tendencias de cumplimiento
- Terminal-style code blocks para detalles técnicos de XML

**Elementos Distintivos:**
1. Líneas de conexión entre elementos relacionados
2. Iconografía de escaneo: líneas horizontales animadas
3. Badges con efecto "glow" para estados críticos

**Filosofía de Interacción:**
- Hover: glow verde suave alrededor del elemento
- Click: efecto de pulso
- Selección: borde verde brillante

**Animación:**
- Escaneo de tabla: línea horizontal que recorre filas
- Cambio de estado: efecto de "destello" en 200ms
- Carga: barras de progreso animadas con efecto de onda

**Sistema Tipográfico:**
- Display: Space Mono Bold 28px para títulos
- Heading: Roboto Mono SemiBold 16px para subtítulos
- Body: Roboto 13px para contenido
- Mono: JetBrains Mono 11px para datos técnicos

## Respuesta 3: Diseño Humanista Accesible (Probabilidad: 0.06)

**Movimiento de Diseño:** Diseño Humanista con énfasis en accesibilidad WCAG AA+

**Principios Centrales:**
- Inclusividad primero: contraste mínimo 7:1
- Amabilidad visual: formas suavizadas, sin aristas agresivas
- Claridad narrativa: guía al usuario a través de la información
- Espaciado adaptativo: responsivo a diferentes tamaños de pantalla

**Filosofía de Color:**
- Paleta cálida: crema (RGB 255, 250, 240) como fondo
- Azul marino (RGB 25, 55, 109) para texto principal
- Verde tierra (RGB 76, 110, 87) para éxito
- Naranja cálido (RGB 217, 119, 6) para advertencias
- Rojo suave (RGB 180, 70, 70) para errores
- Justificación: colores naturales, menos fatiga visual, accesible para daltonismo

**Paradigma de Layout:**
- Flujo vertical natural: encabezado → resumen → tabla → detalles
- Tarjetas con bordes redondeados (16px) y sombra suave
- Espaciado vertical de 24px entre secciones
- Máximo 80 caracteres por línea para legibilidad

**Elementos Distintivos:**
1. Iconografía clara y consistente con etiquetas de texto
2. Indicadores de estado con símbolo + color + texto
3. Paneles informativos con fondo de color suave

**Filosofía de Interacción:**
- Hover: cambio de fondo suave, sin saltos
- Focus: anillo visible de 3px en color primario
- Feedback: toast notifications en esquina inferior derecha

**Animación:**
- Entrada suave: fade-in + slide-up de 400ms
- Transiciones: 250ms para cambios de estado
- Movimiento: easing ease-out para naturalidad

**Sistema Tipográfico:**
- Display: Lora Bold 36px para títulos (serif para calidez)
- Heading: Poppins SemiBold 20px para subtítulos
- Body: Open Sans Regular 15px para contenido (altamente legible)
- Mono: Courier Prime 12px para datos técnicos

---

## Decisión Final

**Elegido: Minimalismo Corporativo Moderno** - Combina precisión fiscal con claridad visual, ideal para auditores y contadores que necesitan información confiable y accesible sin distracciones.
