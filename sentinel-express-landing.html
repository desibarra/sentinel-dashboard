Preguntas frecuentes

Resolvemos tus dudas
antes de empezar.

¿Mis datos fiscales salen de mi PC?
Absolutamente no. Sentinel Express opera 100% offline. Tus XML nunca salen de tu computadora. No hay conexión a la nube para el análisis. La única excepción es la actualización de las listas EFOS del SAT, que descargamos directamente del portal oficial del SAT.
¿Cuántos archivos XML puedo analizar?
No hay límite. Despachos contables han procesado más de 50,000 CFDIs en una sola sesión. El motor está optimizado para manejar cargas masivas sin que tu computadora se congele.
¿Las listas EFOS del SAT están actualizadas?
Sí. Sentinel sincroniza automáticamente con el portal del SAT cada vez que lo abres. Siempre trabajas con la versión más reciente de las listas definitivas, presuntas, desvirtuadas y favorables de EFOS y EDOS.
¿Funciona con CFDI 4.0 y complementos?
Completamente. Sentinel es nativo para CFDI 4.0 e incluye soporte para complementos de carta porte, nómina, pagos, retenciones e información de pagos. También es compatible con CFDI 3.3 para análisis histórico.
¿Tiene API para integrarlo a mi ERP?
Sí. La versión Enterprise incluye API REST documentada en español para integraciones con SAP, CONTPAQi, Aspel, Oracle y ERP a medida. Validación en tiempo real antes de registrar cada factura en tu sistema contable.
¿Qué pasa si encuentro un EFOS?
Sentinel genera un reporte forense detallado con el RFC problemático, los CFDIs afectados, el monto en riesgo y las opciones de acción recomendadas. Este reporte puede servir como evidencia ante el SAT de que actuaste de buena fe al detectar el problema.
